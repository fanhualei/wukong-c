# wukong-c



## 下载代码

```shell
# 选择一个目录
git clone https://github.com/fanhualei/wukong-c.git
cd wukong-c

# 查看当前git的配置
git config --list

# 查看当前目录的文件状态　可以不加-s 这样提示会更多
git status -s

# 提交到本地
git commit -m "注释的内容"

# 跳过使用暂存区域 给 git commit 加上 -a 选项，
git commit -m -a "注释的内容"

# 推送到远程仓库
git push

# 查看远程服务器的
git remote show
```
